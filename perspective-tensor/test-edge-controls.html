Edge Controls Test Checklist

Visual Controls to Verify:

  1. Edge Width Slider
    • Range: 0.5 to 3.0
    • Shows current value
    • Has tick marks at 0.5, 1.0, 1.5, 2.0, 2.5, 3.0
  2. Edge Style Dropdown
    • Options: Solid, Dashed, Dotted
    • Changes edge rendering pattern
    • Styled with dark theme (black background, white text)
  3. Edge Color Picker
    • Separate colors for Dark and Light themes
    • Auto button for light theme
    • Shows current color value
  4. Edge Opacity Slider
    • Range: 0% to 100%
    • Shows percentage value
  5. Show Edges Checkbox
    • Toggles edge visibility

Persistence Tests:

  1. Change edge style to "Dashed" → Refresh page → Should remain "Dashed"
  2. Set edge width to 2.0 → Switch shapes → Edge width should stay 2.0
  3. Change edge color → Switch theme → Color should update appropriately
  4. Disable edges → Change shape → Edges should remain hidden

Visual Tests:

  1. Solid edges: Continuous lines
  2. Dashed edges: Lines with gaps
  3. Dotted edges: Small dots with larger gaps
  4. Edge opacity affects transparency
  5. Edge color changes apply immediately
